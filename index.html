Dibuat oleh: Kelompok 5 Kelas E Informatika

Data Input

Definisi Fungsi

f(x) = x - diskon_nominal → Fungsi diskon nominal
g(x) = x * (1 - diskon_persen/100) → Fungsi diskon persentase
h(x) = x * (1 + pajak/100) → Fungsi penambahan pajak

Hasil Analisis

(f ∘ g)(x) = f(g(x))

(g ∘ f)(x) = g(f(x))
Rp 0
